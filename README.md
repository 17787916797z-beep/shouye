# 瑶音闯关 - 蓝靛瑶文化展示网站

一个致力于守护蓝靛瑶千年文化的数字化展示平台，通过游戏化方式传播瑶族语言和文化。

## 网站特色

- 🎨 精美的响应式设计，支持移动端和桌面端
- 📚 完整的蓝靛瑶文化展示（盘王鼓、蓝靛染布、芦笙对歌等）
- 🎮 游戏化学习平台入口
- 📖 非遗传承人故事介绍
- 💬 用户反馈展示

## 技术栈

- HTML5
- Tailwind CSS (CDN)
- Font Awesome 图标库

## 如何查看

直接在浏览器中打开 `index.html` 文件即可查看网页。

## 项目背景

本项目旨在数字化保护与活态传承蓝靛瑶非物质文化遗产，让更多人了解瑶族文化，让年轻一代重新爱上自己的母语。
